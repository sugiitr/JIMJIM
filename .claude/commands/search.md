# /search — プロジェクト全体を検索する

プロジェクト内の全ファイルから指定したキーワード・パターン・正規表現を検索する。影響範囲調査に便利。

## 手順

1. `Grep` ツールで対象パターンを検索する
2. ヒットしたファイル・行番号を一覧表示する
3. 関連箇所の文脈を示す
4. 影響範囲のサマリーを報告する

## 出力形式

```
## 検索結果: "キーワード"
- `src/foo.ts:42` — 該当行の内容
- `src/bar.ts:18` — 該当行の内容

## 影響範囲: X件
```

$ARGUMENTS に検索キーワードまたは正規表現を渡してください。
