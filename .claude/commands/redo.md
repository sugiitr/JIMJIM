# /redo — 取り消した変更を再適用する

`/undo` で取り消した変更、または以前のバージョンの変更を再適用する。

## 手順

1. `git log` でコミット履歴を確認する
2. `git stash list` でスタッシュを確認する
3. 再適用したい変更を特定する
4. ユーザーに確認してから再適用する

## ルール

- 確認なしに実行しない
- コンフリクトが起きる可能性を事前に警告する
- スタッシュから復元 or `git cherry-pick` で対応

## コマンド例

```bash
# スタッシュから復元
git stash pop

# 特定のコミットを再適用
git cherry-pick <コミットID>

# 取り消したコミットを再適用
git revert <revertコミットID>
```

$ARGUMENTS に再適用したいコミットID・スタッシュ番号を渡してください（省略時は直近のスタッシュ）。
