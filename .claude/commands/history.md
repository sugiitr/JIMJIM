# /history — 変更履歴を確認する

Gitの変更履歴を見やすく表示し、何がいつ変更されたかを把握する。

## 手順

1. `git log` で変更履歴を取得する
2. 指定されたファイル・範囲に絞り込む
3. 変更内容を分かりやすく整理して表示する
4. 注目すべき変更・問題点を指摘する

## 表示モード

- **全体履歴**: `git log --oneline -20`
- **ファイル別**: `git log --oneline -- <ファイルパス>`
- **差分付き**: `git log -p --follow <ファイルパス>`
- **グラフ**: `git log --oneline --graph --decorate`

## 出力形式

```
## 変更履歴

| コミットID | 日時 | 変更内容 |
|-----------|------|---------|
| abc1234 | 〇月〇日 | feat: 〜 |
| def5678 | 〇月〇日 | fix: 〜 |

## 注目ポイント
- 〜
```

$ARGUMENTS にファイルパス・件数・期間を渡してください（省略時は直近20件）。
