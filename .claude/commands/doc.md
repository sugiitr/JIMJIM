# /doc — ドキュメントを生成する

対象コードに対して、関数の説明・引数・戻り値などのドキュメント（JSDoc / TSDoc）を自動追加する。

## 手順

1. 対象ファイル・関数を読み込む
2. 関数の目的・引数・戻り値・例外を分析する
3. JSDoc/TSDoc形式でコメントを生成する
4. コードに追加する

## 出力例

```typescript
/**
 * ユーザーを認証する
 * @param email - メールアドレス
 * @param password - パスワード
 * @returns 認証済みユーザーオブジェクト
 * @throws {AuthError} 認証失敗時
 */
```

$ARGUMENTS に「ファイル名」または「クラス名・関数名」を渡してください。
例: `/doc src/auth.ts` または `/doc このクラスに`
