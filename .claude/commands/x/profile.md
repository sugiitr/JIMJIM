# /x/profile — Xプロフィールを最適化する

フォロー率を上げる、最強のXプロフィール文を生成する。

## 手順

1. 現在のプロフィール（あれば）と自己情報を確認する
2. プロフィールの役割を整理する（誰向け・何者・何を発信）
3. 複数のプロフィール案を生成する
4. A/Bテスト推奨パターンを提示する

## プロフィールの構造

```
1行目: 何者か（ポジション・実績）
2行目: フォローするメリット（何が得られるか）
3行目: 発信内容（テーマ・頻度）
4行目: 人間性・個性（信頼・親近感）
最後: CTA（フォロー・リンク誘導）
```

## プロフィールの禁忌

- 趣味の羅列（誰も興味ない）
- 「〇〇勉強中」（信頼を下げる）
- 「フォロバ100%」（質を下げる）
- 何者かわからない抽象表現

## 出力形式

```
## プロフィール案: 「〜」ポジション

### シンプル版 (100字以内)
〜

### 詳細版 (160字)
〜

### 実績強調版
〜
```

$ARGUMENTS に自分の実績・発信ジャンル・ターゲット層・個性を渡してください。
