# /x/plan — X/Twitterの投稿計画を立てる

週間・月間の投稿スケジュール・コンテンツカレンダーを設計する。

## 手順

1. 目標・ターゲット・発信ジャンル・頻度を確認する
2. 投稿タイプのバランスを設計する（有益/共感/セールス）
3. 週間コンテンツカレンダーを生成する
4. 各投稿のテーマ・型・目的を割り当てる

## 投稿タイプのバランス（黄金比）

```
有益投稿（教育・ノウハウ）: 60%
共感投稿（日常・感情・意見）: 30%
セールス投稿（告知・誘導）: 10%
```

## 最適投稿時間（日本標準）

- 平日朝: 7:00〜9:00
- 昼休み: 12:00〜13:00
- 夜: 21:00〜23:00

## 出力形式

```
## 投稿計画: 〇月〇週 / 〜テーマ

| 曜日 | 時間 | タイプ | テーマ | 型 |
|------|------|--------|--------|-----|
| 月 | 7:00 | 有益 | 〜 | リスト |
| 水 | 12:00 | 共感 | 〜 | ストーリー |
...
```

$ARGUMENTS に目標・ジャンル・週の投稿数を渡してください。
