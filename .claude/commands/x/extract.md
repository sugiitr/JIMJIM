# /x/extract — X/Twitterから情報を抽出する

投稿・スレッド・アカウントから価値ある情報・インサイト・データを抽出する。

## 手順

1. 対象を確認する（投稿URL / スレッド / アカウント名）
2. 抽出する情報の種類を特定する
3. 構造化してデータを整理する
4. 活用方法を提案する

## 抽出できる情報

- 数値・統計データ
- 引用・名言
- リスト（ツール・書籍・サービス等）
- ノウハウ・ハック
- 連絡先・リソースリンク

## 出力形式

```
## 抽出結果: 「〜」

### 抽出データ
| 項目 | 内容 |
|------|------|
| 〜 | 〜 |

### 活用アイデア
- 〜
```

$ARGUMENTS に抽出元のURL・テキスト・「抽出したい情報の種類」を渡してください。
