# /x/deconstruct — バズ投稿を完全解体する

バズった投稿の構造・心理・テクニックを徹底的に分解して学ぶ。

## 手順

1. 対象投稿（URL/テキスト）を確認する
2. 構造レイヤー（表層→深層）で分解する
3. 使われているテクニックをすべて抽出する
4. 再現するための「型」を生成する

## 分解レイヤー

```
【表層】テキスト・フォーマット・長さ
↓
【構造】フック→本論→締めの設計
↓
【感情】使われている感情トリガー
↓
【心理】読者の思考・行動の誘導
↓
【タイミング】なぜ今バズったか
```

## 出力形式

```
## 解体レポート: 「〜」

### 基本スペック
- 文字数: 〜
- いいね: 〜 / RT: 〜

### 構造分析
フック: 「〜」→ 技: 〜
本論: 〜 → 技: 〜
締め: 〜 → 技: 〜

### 使用テクニック一覧
1. 〜
2. 〜

### 再現テンプレート
「〜[自分の内容]〜」
```

$ARGUMENTS に解体したい投稿のURL・テキストを渡してください。
