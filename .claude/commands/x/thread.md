# /x/thread — X/Twitterスレッドを作成する

1つのテーマを深掘りする、読まれるスレッド（連ツイ）を作成する。

## 手順

1. テーマ・目的・ターゲットを確認する
2. 全体構成（ツイート数・流れ）を設計する
3. 各ツイートを生成する
4. 流れ・一貫性を確認してブラッシュアップする

## スレッド構成の型

```
1/n: フック（最強の一言）
2/n: 問題提起・共感
3/n〜n-1/n: 本論（学び・ノウハウ・ストーリー）
n/n: まとめ + CTA（フォロー・保存・シェア誘導）
```

## ルール

- 各ツイートは140字以内
- 各ツイートに「次を読む理由」を作る
- 最後のツイートで行動を促す

## 出力形式

```
1/〇 〜
---
2/〇 〜
---
3/〇 〜
---
〇/〇 まとめ: 〜
→ 参考になったら保存＆フォローお願いします
```

$ARGUMENTS にスレッドのテーマ・ツイート数（省略時は5〜7）を渡してください。
