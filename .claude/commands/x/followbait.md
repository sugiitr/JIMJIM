# /x/followbait — フォローされる投稿を作る

「この人をフォローしたい！」と思わせる、フォロワー獲得に特化した投稿を生成する。

## 手順

1. 自分のアカウントの強み・発信内容を確認する
2. フォローする「理由」を明確にする投稿を設計する
3. 「今後も見たい」と感じさせる価値提示をする
4. フォロー誘導を自然に組み込む

## フォローが生まれる投稿の条件

- 「このアカウント、他にもこんな情報出してるんだ」と思わせる
- プロフに飛びたくなる好奇心を刺激する
- 「定期的に見たい」コンテンツを予告する
- 「自分の役に立つ」と感じさせる実用的な内容

## 型

```
【価値投稿 → フォロー誘導型】
有益な情報を出す
→「こういう情報を定期投稿してます」
→「プロフからフォローどうぞ」

【自己紹介 + 宣言型】
「私は〜です。毎週〇回、〜について発信します」
「フォローすると〜がわかります」
```

## 出力形式

```
## フォローbait投稿: 「〜」強み

### 価値先出し型
〜

### 自己紹介宣言型
〜

### プロフィール誘導型
〜
```

$ARGUMENTS に自分の強み・発信内容・ターゲット層を渡してください。
