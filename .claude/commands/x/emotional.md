# /x/emotional — 感情を動かす投稿を作る

読者の感情（感動・共感・希望・怒り）を揺さぶる投稿を生成する。

## 手順

1. 伝えたい感情・テーマを確認する
2. 感情を最大化する表現・構造を設計する
3. 感情トリガーを組み込んだ投稿を生成する
4. 感情の強度と投稿の安全性を確認する

## 感情トリガー別アプローチ

| 感情 | トリガー | 型 |
|------|---------|-----|
| 感動 | 自己犠牲・成長・再会 | ストーリー |
| 共感 | 失敗・孤独・葛藤 | あるある |
| 希望 | 変化・可能性・before/after | 転換 |
| 怒り | 不公平・搾取・理不尽 | 告発 |
| 驚き | 反直感・反常識・数字 | 暴露 |

## 出力形式

```
## 感情訴求投稿: 「〜」ターゲット感情

### 狙う感情: 〜
### 使用するトリガー: 〜

### 投稿案
〜

### 感情のクライマックスはどこか
→ 「〜」の部分
```

$ARGUMENTS にテーマ・狙いたい感情（感動/共感/希望/怒り/驚き）を渡してください。
