# /x/section — コンテンツの特定セクションを書く

アウトラインの中の1つのセクション・ツイートを集中して仕上げる。

## 手順

1. 全体アウトラインとターゲットセクションを確認する
2. セクションの役割（導入/本論/まとめ）を把握する
3. 前後との流れを意識してセクションを生成する
4. 複数バリエーションを提示する

## ルール

- セクション単体で完結しつつ、前後とつながる
- 冗長な繰り返しを避ける
- 読者が「もっと読みたい」と思う終わり方にする

## 出力形式

```
## セクション執筆: 「〜」

### 文脈
- 前のセクション: 〜
- このセクションの役割: 〜
- 次のセクション: 〜

### 生成結果

#### バージョン1
〜

#### バージョン2
〜
```

$ARGUMENTS にアウトライン全体とターゲットセクション番号・内容を渡してください。
