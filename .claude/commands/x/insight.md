# /x/insight — 独自インサイトを発掘する

データ・体験・観察から「他の人が言わない独自の気づき」を言語化する。

## 手順

1. 素材（データ・体験・観察・感覚）を確認する
2. 「当たり前だと思われていること」への疑問を立てる
3. 反直感的・非自明なインサイトを5〜10個生成する
4. 各インサイトを投稿の冒頭1行に変換する

## インサイト発掘の問い

- 「なぜ逆のことが起きるのか？」
- 「みんなが信じているが実は違うことは？」
- 「Aが原因ではなくBが原因では？」
- 「誰も注目していないが重要なことは？」
- 「10年後には常識になることは？」

## 出力形式

```
## インサイト発掘: 「〜」テーマ

### 発掘したインサイト

**インサイト1**
「〜」
→ なぜこれが真実: 〜
→ 投稿フック: 「〜」

**インサイト2**
「〜」
...

### 投稿に使えるTOP3
1. 〜（バズ可能性: ★★★★）
```

$ARGUMENTS に分析したいテーマ・データ・体験を渡してください。
