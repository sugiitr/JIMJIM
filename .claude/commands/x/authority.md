# /x/authority — 権威・専門性を示す投稿を作る

「この人は本物だ」「信頼できる」と感じさせる、権威付け投稿を生成する。

## 手順

1. 自分の実績・専門性・経験を確認する
2. 「証拠」となる要素を整理する
3. 権威を自然にアピールする投稿を生成する
4. 押し付けがましくなく、事実で語る構成にする

## 権威を示す要素

- **数字**: 実績・規模・年数・件数
- **固有名詞**: 所属・クライアント・媒体
- **結果**: before/after・成果
- **認定**: 資格・表彰・掲載
- **経験**: 「〇年で〇回〜した」

## 権威投稿の原則

- 自慢ではなく「証拠」として提示する
- 謙虚さを忘れない（「まだ途中ですが」等）
- 具体的な数字で信頼を担保する
- 「だからこう思う」と意見に繋げる

## 出力形式

```
## 権威付け投稿: 「〜」実績

### 実績提示型
〜

### ストーリー型（自慢に見えない）
〜

### 数字で語る型
〜
```

$ARGUMENTS に実績・専門性・経験を渡してください。
