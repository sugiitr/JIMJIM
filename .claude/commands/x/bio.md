# /x/bio — 自己紹介投稿を作る

ピン留めや定期投稿に使える、魅力的な自己紹介投稿を生成する。

## 手順

1. 自己情報（経歴・実績・発信内容・価値観）を確認する
2. 「この人に興味が湧く」自己紹介の構成を設計する
3. 単ツイート版・スレッド版を生成する
4. ピン留め用・通常投稿用を区別する

## 刺さる自己紹介の要素

- **意外性**: 「え、そんな経歴？」
- **実績**: 具体的な数字・成果
- **ストーリー**: before/after
- **価値提示**: フォローすると何が得られるか
- **人間性**: 失敗談・趣味・パーソナリティ

## 自己紹介の型

```
「〜な経歴を持つ〜です」（意外性・実績）
「〜を経験して気づいたこと：〜」（ストーリー）
「ここでは〜について発信しています」（価値提示）
「〜な方はぜひフォロー」（ターゲット指名）
```

## 出力形式

```
## 自己紹介投稿: 「〜さん」

### ピン留め用（スレッド 5ツイート）
1/5: 〜
...

### 単ツイート版（140字）
〜
```

$ARGUMENTS に経歴・実績・発信内容・ターゲット・個性を渡してください。
