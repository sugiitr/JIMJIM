# /x/gap — 競合が語っていない隙間を見つける

自分のジャンルで、他のアカウントが発信していない「空白地帯」を発見する。

## 手順

1. 発信ジャンル・競合アカウントを確認する
2. 競合が多く語っているトピックをリストアップする
3. 語られていない・少ないトピックを特定する
4. 自分が占めるべき空白地帯を定義する

## ギャップの種類

- **テーマのギャップ**: 誰も語っていない話題
- **視点のギャップ**: 同じ話題でも違う立場から
- **フォーマットのギャップ**: 違う形式（スレッド/画像等）
- **ターゲットのギャップ**: 誰も向いていない層
- **深さのギャップ**: 表面的な話ばかりで深掘りなし

## 出力形式

```
## ギャップ分析: 「〜」ジャンル

### 競合が多く語っている（飽和状態）
- 〜（競合アカウント数: 多）

### 語られていない空白地帯
| ギャップ | 種類 | 機会の大きさ |
|---------|------|------------|
| 〜 | テーマ | ★★★★ |

### あなたが占めるべきポジション
〜
```

$ARGUMENTS に自分の発信ジャンル・競合アカウント名を渡してください。
