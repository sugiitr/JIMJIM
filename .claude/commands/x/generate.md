# /x/generate — バズるX投稿を生成する

指定したテーマ・ターゲット・目的に合わせて、バズりやすいX投稿を生成する。

## 手順

1. テーマ・ターゲット・投稿目的を確認する
2. 複数のアングル・フックパターンを検討する
3. 3〜5パターンの投稿文を生成する
4. 各パターンの狙い・特徴を解説する

## バズるフックの型

- 数字系: 「〇〇するだけで〇〇が変わる」
- 逆説系: 「〇〇は間違いだった」
- 暴露系: 「誰も言わない〇〇の真実」
- 共感系: 「〇〇な人、全員集合」
- 疑問系: 「なぜ〇〇する人は〇〇なのか？」

## 出力形式

```
## 生成投稿: 「〜」テーマ

### パターン1（数字系）
〜

### パターン2（逆説系）
〜

### パターン3（共感系）
〜
```

$ARGUMENTS に投稿のテーマ・ターゲット読者・目的（認知/集客/販売）を渡してください。
