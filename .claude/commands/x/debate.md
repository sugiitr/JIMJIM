# /x/debate — 議論を呼ぶ投稿を作る

意見が分かれ、返信・引用ツイートが殺到する「議論系投稿」を生成する。

## 手順

1. テーマ・立場を確認する
2. 賛否が分かれる言い切り文を設計する
3. 反論を予測し、それに耐えられる構造にする
4. 炎上せず議論になる最適な強度を調整する

## 議論を呼ぶ投稿の特徴

- 「Yes / No」で答えられる命題
- 「え、そうなの？」と思わせる逆説
- 一方を肯定することで他方が否定される構造
- 正解がなく主観的な意見を問う内容
- 「自分の考えを言いたい」感情を刺激

## 強度調整

- 弱: 「〜派ですか？」（穏やか）
- 中: 「〜は間違いだと思う」（意見系）
- 強: 「〜している人は〜だ」（挑発系）
→ **中程度を基本に**（炎上リスクを避ける）

## 出力形式

```
## 議論系投稿: 「〜」テーマ

### 弱（穏やか）
〜

### 中（推奨）
〜

### 強（リスク有り）
〜

### 予想される反論
- 〜 → 返し: 〜
```

$ARGUMENTS に議題・自分の立場を渡してください。
