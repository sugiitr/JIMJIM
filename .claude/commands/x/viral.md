# /x/viral — バイラルになる投稿を設計する

シェア・引用・拡散されやすい投稿を設計・生成する。

## 手順

1. テーマ・拡散させたい層を確認する
2. バイラル要素（感情・有用性・希少性）を分析する
3. 拡散しやすい構造の投稿を設計する
4. 投稿タイミング・ハッシュタグ戦略を提案する

## バイラルの6要素

1. **感情的高揚**: 驚き・感動・怒り
2. **実用性**: 今すぐ使える情報
3. **希少性**: 他では見られない視点
4. **アイデンティティ**: 「自分らしい」と思える内容
5. **社会的証明**: 数字・実績・エビデンス
6. **フォーマット**: リスト・図解・スレッド

## 出力形式

```
## バイラル設計: 「〜」

### 拡散ターゲット
〜

### 使用するバイラル要素
- 主: 〜
- 副: 〜

### 投稿案
〜

### 投稿戦略
- 最適時間: 〜
- ハッシュタグ: #〜 #〜
- 初動施策: 〜
```

$ARGUMENTS に拡散させたいテーマ・ターゲット層を渡してください。
