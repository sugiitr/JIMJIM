# /x/story — ストーリー型投稿を作る

自己体験・事例・変化のプロセスをストーリー形式の投稿に変換する。

## 手順

1. ストーリーの素材（体験・事例・変化）を確認する
2. ストーリー構造（起承転結）を設計する
3. 感情的な引きを最大化した投稿を生成する
4. スレッド版・単ツイート版を両方作成する

## ストーリーの構造

```
【起】 状況設定（主人公・時間・場所）
       ↓「普通の日常が始まる」
【承】 葛藤・問題の発生
       ↓「何かが変わり始める」
【転】 転換点・気づき
       ↓「ここで主人公が変わる」
【結】 結末・学び・CTA
       ↓「読者への問いかけ」
```

## 感情トリガー

- 共感: 「私も同じでした」
- 驚き: 「まさかこんな結果に」
- 希望: 「〇ヶ月で人生が変わった」

## 出力形式

```
## ストーリー投稿: 「〜」

### 単ツイート版
〜

### スレッド版（4ツイート）
1/4: 〜
2/4: 〜
3/4: 〜
4/4: 〜
```

$ARGUMENTS にストーリーの素材・体験談・変化のポイントを渡してください。
