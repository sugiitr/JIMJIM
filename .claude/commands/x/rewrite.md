# /x/rewrite — X投稿をバズる文章に書き直す

既存の投稿・文章をバズりやすい表現・構造に書き直す。

## 手順

1. 元の投稿を読み込み、核心メッセージを抽出する
2. 弱い点（フック・構造・感情・CTA）を特定する
3. 改善版を複数パターン生成する
4. 変更点と改善理由を解説する

## チェックポイント

- 冒頭1行で読み続けさせているか？
- 読者が「自分のことだ」と思えるか？
- 具体的な数字・固有名詞を使っているか？
- 最後に行動を促しているか？
- 改行・句読点で読みやすいか？

## 出力形式

```
## 元の投稿
〜

## 問題点
- 〜

## 改善版
### バージョン1
〜

### バージョン2（より強いフック）
〜
```

$ARGUMENTS に書き直したい元の投稿テキストを渡してください。
