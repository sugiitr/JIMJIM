# /x/shock — 驚かせる投稿を作る

「え！？」「まじで！？」と思わせる、驚き・衝撃系投稿を生成する。

## 手順

1. テーマ・驚かせたい内容を確認する
2. 「反直感的な事実」「意外な数字」「知られていない真実」を掘り起こす
3. 驚きを最大化する構成・言葉を選ぶ
4. 信頼性も担保した投稿に仕上げる

## 驚き投稿の公式

```
【事実提示】「〇〇は実は〜だった」
   ↓
【驚きの強化】「つまり〜ということ」
   ↓
【根拠】「〜によれば」「私の経験では」
   ↓
【示唆】「だから〜する必要がある」
```

## 驚きの種類

- **数字系**: 「〜は実は〇〇%だった」
- **逆転系**: 「〜だと思っていたが、実は〜だった」
- **知識系**: 「〇〇の本当の意味は〜」
- **スケール系**: 「〜すると〇年後には〜になる」

## 出力形式

```
## 衝撃系投稿: 「〜」テーマ

### 驚きのネタ
〜

### 投稿案（強め）
〜

### 投稿案（ファクトベース）
〜
```

$ARGUMENTS に驚きを作りたいテーマ・ジャンルを渡してください。
