# /x/draft — コンテンツの下書きを作る

アウトラインや指示をもとに、X投稿・スレッド・ノートの下書きを生成する。

## 手順

1. テーマ・アウトライン（あれば）・トーンを確認する
2. 指定形式で全文を一気に下書きする
3. 複数バリエーションを生成する
4. 磨き方のポイントを提示する

## ルール

- 下書きはあえて「少し多め」に書く（削る前提）
- 冒頭と締めは特に力を入れる
- 自分の言葉・体験談を入れるプレースホルダーを示す
- 「[実体験を入れる]」「[数字を入れる]」などの指示を含める

## 出力形式

```
## 下書き: 「〜」

### バージョン1
〜

[実体験を入れる: 〜について自分の経験を1文]

〜

### バージョン2（トーン違い）
〜

### 磨きポイント
- 〜
```

$ARGUMENTS にテーマ・形式・トーン・文字数目安を渡してください。
