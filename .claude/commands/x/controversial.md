# /x/controversial — 常識を覆す投稿を作る

「え、そんなこと言っていいの？」という、常識に反する意見投稿を生成する。

## 注意事項

**差別・誹謗中傷・不法行為は絶対にしない。**
常識を覆すのは「思考の転換」であり「他者攻撃」ではない。

## 手順

1. テーマ・業界の「当たり前」を確認する
2. それを否定または反転させる命題を作る
3. 説得力のある根拠を添える
4. 読者が「確かに…」と思える論理構成にする

## 「逆張り」の安全な型

```
「〜は間違いだ（思考の転換）」
→「なぜなら〜（論理的根拠）」
→「実際に〜（実例・体験）」
→「本当は〜すべきだ（提案）」
```

## 出力形式

```
## 常識覆し投稿: 「〜」テーマ

### ターゲット「当たり前」
〜

### 反転命題
〜

### 投稿案
〜

### 根拠・補強材料
- 〜
```

$ARGUMENTS に覆したい常識・テーマを渡してください。
