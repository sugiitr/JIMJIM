# /x/replicate — バズ投稿を自分流に再現する

バズった投稿の構造・パターンを使い、自分のテーマで新しい投稿を作る。

## 手順

1. 参考にするバズ投稿と自分のテーマを確認する
2. バズ投稿の「型」を抽出する
3. 型に自分のコンテンツを当てはめる
4. 完成度を確認してブラッシュアップする

## 再現の原則

- コピーではなく「構造」を借りる
- 内容は完全にオリジナルにする
- 自分の実体験・数字・固有名詞に置き換える
- 読者層に合わせてトーンを調整する

## 出力形式

```
## 再現作業

### 参考にした投稿
「〜」（@〜）

### 抽出した型
フック: 〜
構造: 〜
締め: 〜

### 自分流バージョン（テーマ: 〜）
〜

### 変更した要素
- 〜 → 〜
```

$ARGUMENTS に参考バズ投稿と自分のテーマを渡してください。
