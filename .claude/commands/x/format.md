# /x/format — 投稿のフォーマットを整える

改行・句読点・絵文字・記号を最適化し、スマホで読みやすい投稿に整形する。

## 手順

1. 元の投稿を読み込む
2. 現在のフォーマット問題を特定する
3. 読みやすさを最大化した整形版を生成する
4. 変更点と理由を解説する

## フォーマットの原則

- **改行**: 2〜3文ごとに空行を入れる
- **文章の長さ**: 1文20〜40字が読みやすい
- **絵文字**: 適度に使うとスキャンしやすい（1〜3個/投稿）
- **箇条書き**: 3つ以上の項目はリスト化
- **ハッシュタグ**: 末尾に2〜3個、本文と分ける
- **記号**: 「→」「▶」「▷」で視線誘導

## スマホ最適化

- 1行20字前後を意識
- 上部3行で判断される（fold above）
- 縦長より横展開

## 出力形式

```
## 整形前
〜

## 整形後
〜

## 変更点
- 〜
```

$ARGUMENTS に整形したい投稿テキストを渡してください。
