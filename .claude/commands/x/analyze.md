# /x/analyze — X/Twitterの投稿・アカウントを分析する

指定した投稿・スレッド・アカウントを詳細分析し、バズった要因・戦略を解明する。

## 手順

1. 対象URL・テキスト・アカウントを確認する
2. 構造・トーン・フック・CTA・エンゲージメント要素を分析する
3. バズった理由・効いた要素を特定する
4. 改善点・応用アイデアを提示する

## 分析観点

- **フック**: 冒頭1行の引力
- **構造**: 情報の流れ・段落分け
- **感情トリガー**: 共感・怒り・驚き・希望
- **CTA**: アクション誘導の有無と強度
- **タイミング**: 投稿時間・トレンドとの関係

## 出力形式

```
## 分析結果: @〜 / 「〜」
### バズった要因 (TOP3)
1. 〜
2. 〜
3. 〜

### 構造分析
- フック: 〜
- 本文構成: 〜
- 締め: 〜

### 応用アイデア
- 〜
```

$ARGUMENTS に分析したい投稿URL・テキスト・アカウント名を渡してください。
