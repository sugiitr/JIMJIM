# /x/compare — X/Twitterの投稿・アカウントを比較する

複数の投稿・アカウント・戦略を比較し、優劣・差異・強みを明確にする。

## 手順

1. 比較対象を確認する（最低2つ）
2. 比較軸を設定する
3. 各要素を評価・採点する
4. 総合判断とおすすめを提示する

## デフォルト比較軸

- エンゲージメント率
- フックの強さ
- 情報の価値
- 読みやすさ
- CTA効果

## 出力形式

```
## 比較: A vs B

| 比較軸 | A | B |
|--------|---|---|
| エンゲージメント | ★★★★ | ★★★ |
| フック強度 | 〜 | 〜 |

### 勝者: 〜
理由: 〜

### 各自の強み
- A: 〜
- B: 〜
```

$ARGUMENTS に比較したい2つ以上の投稿URL・アカウント・テキストを渡してください。
