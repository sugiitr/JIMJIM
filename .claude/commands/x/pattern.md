# /x/pattern — バズパターンを抽出する

複数のバズ投稿から共通パターン・法則を抽出する。

## 手順

1. 分析対象の投稿群またはアカウントを確認する
2. 共通要素（構造・フック・感情・タイミング）を抽出する
3. パターンを体系化してカテゴリ分けする
4. パターンを自分の発信に応用する方法を提示する

## 抽出する共通要素

- **開始パターン**: フックの型
- **長さパターン**: 最適な文字数・ツイート数
- **感情パターン**: 最も反応が多い感情
- **時間パターン**: バズりやすい投稿時間
- **構造パターン**: リスト/ストーリー/問答

## 出力形式

```
## バズパターン分析: 「〜」ジャンル

### 発見したパターン (N個)

**パターン1: 〜**
- 出現頻度: 〇/〇投稿
- 特徴: 〜
- 例: 「〜」

**パターン2: 〜**
...

### 最強パターンTOP3
1. 〜（理由: 〜）

### 応用方法
〜
```

$ARGUMENTS に分析したいアカウント名・投稿群・ジャンルを渡してください。
