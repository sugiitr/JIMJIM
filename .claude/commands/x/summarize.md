# /x/summarize — X/Twitterの情報をまとめる

特定トピックのX上の議論・トレンドを要約し、要点を整理する。

## 手順

1. 対象トピック・期間を確認する
2. 主要な意見・論点を収集・整理する
3. 賛否・多数派・少数派の意見を分類する
4. エグゼクティブサマリーを生成する

## 出力形式

```
## 「〜」X上のまとめ

### 概要 (3行以内)
〜

### 主要な意見
**ポジティブ派**
- 〜

**批判派**
- 〜

### 注目ツイート・アカウント
- @〜: 「〜」(いいね: 〜)

### 結論
〜
```

$ARGUMENTS にまとめたいトピック・ハッシュタグ・URLを渡してください。
