# /x/title — クリックされるタイトルを生成する

ノート・記事・スレッドのタイトルを、クリック率が高い形で複数生成する。

## 手順

1. コンテンツの内容・ターゲット・目的を確認する
2. 複数のタイトル公式を適用する
3. 10〜15個のタイトル案を生成する
4. 各タイトルのクリック率予測と理由を解説する

## タイトル公式

1. **数字 + ベネフィット**: 「〇つの〜で〜が変わる」
2. **How to**: 「〜する方法」「〜のやり方」
3. **Why**: 「なぜ〜なのか」
4. **秘密暴露**: 「〜の真実」「誰も教えてくれない〜」
5. **警告**: 「〜してはいけない理由」
6. **疑問形**: 「〜できていますか？」
7. **対比**: 「〜vs〜」「〜と〜の違い」
8. **ターゲット指名**: 「〜な人だけ読んで」

## 出力形式

```
## タイトル生成: 「〜」コンテンツ

1. 〜（数字系 ★★★★）
2. 〜（秘密暴露系 ★★★）
...

### おすすめTOP3
〜
```

$ARGUMENTS にコンテンツの概要・ターゲット読者を渡してください。
