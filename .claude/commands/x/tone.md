# /x/tone — 投稿のトーンを変換する

同じ内容のまま、投稿のトーン・口調・雰囲気を変換する。

## 手順

1. 元の投稿とターゲットトーンを確認する
2. トーンの特徴・語彙・リズムを定義する
3. 指定トーンで全文を書き直す
4. トーン変化による効果を解説する

## トーン一覧

| トーン | 特徴 | 適した場面 |
|--------|------|-----------|
| カジュアル | 友達感、口語 | 共感・日常投稿 |
| 専門家 | 権威・データ重視 | 教育・信頼構築 |
| 熱狂的 | 感情全開、エネルギー | 新発売・イベント |
| 落ち着き | 丁寧・知的 | ブランド・思想系 |
| 挑発的 | 逆張り・問いかけ | 議論・バズ狙い |
| 共感的 | 寄り添い・優しさ | 悩み解決・DM誘導 |

## 出力形式

```
## トーン変換: 〜 → 〜

### 元の投稿
〜

### 変換後（〜トーン）
〜

### トーンの特徴を反映した部分
- 〜
```

$ARGUMENTS に元の投稿テキストとターゲットトーン名を渡してください。
