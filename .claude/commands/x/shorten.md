# /x/shorten — 投稿を短く・鋭く削る

長い文章や冗長な投稿を、インパクトを保ちながら140字以内に圧縮する。

## 手順

1. 元の文章を読み込み、核心メッセージを1文で抽出する
2. 削除できる要素（修飾語・重複・回りくどさ）を特定する
3. 複数パターンの短縮版を生成する
4. 短縮前後の比較と改善点を解説する

## 削り方の原則

- 「〜ということ」「〜ではないかと」→ 断言に変換
- 接続詞の多用 → 削除して余韻で繋ぐ
- 形容詞の重ね → 最強の1つに絞る
- 主語の繰り返し → 省略
- 「〜だと思います」→ 「〜だ」

## 出力形式

```
## 元の文章 (〇字)
〜

## 短縮版

### パターン1 (〇字)
〜

### パターン2 (〇字、より鋭く)
〜

## 削った要素
- 〜 → 削除理由
```

$ARGUMENTS に短くしたい文章を渡してください。
