# /x/critic — 投稿に辛口フィードバックを出す

遠慮なく問題点を指摘し、バズらない理由を明確にする。褒めない。直す。

## 手順

1. 投稿を読み込む
2. 「なぜバズらないか」を徹底的に分析する
3. 問題点を優先度順に列挙する
4. 各問題の改善策を具体的に提示する

## 批評の観点

1. **フックの弱さ**: 1行目で離脱していないか
2. **ターゲットのぶれ**: 誰向けか不明瞭でないか
3. **価値の薄さ**: 「だからなに？」と思われないか
4. **構成の迷子**: 何を言いたいか伝わるか
5. **言葉の弱さ**: 断言・具体性が不足していないか
6. **差別化の欠如**: 誰でも書けそうでないか

## 出力形式

```
## 辛口批評: 「〜」

### 総合評価: D（バズらない）

### 致命的問題
1. 〜 → 直し方: 〜

### 中程度の問題
2. 〜 → 直し方: 〜

### 改善後のポテンシャル
〜（直せばBランク相当）
```

$ARGUMENTS にレビューしてほしい投稿テキストを渡してください。
