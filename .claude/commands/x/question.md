# /x/question — 返信を呼ぶ質問投稿を作る

フォロワーが思わず返信したくなる質問・アンケート投稿を生成する。

## 手順

1. テーマ・ターゲット層を確認する
2. 答えやすく・意見が分かれる質問を設計する
3. 複数パターンの質問投稿を生成する
4. エンゲージメント予測と理由を解説する

## 返信が来る質問の条件

- 答えが2択〜4択で答えやすい
- 「自分の意見を言いたい」気持ちになる
- 正解がない（論争になりにくい意見系）
- テーマがフォロワーの日常・関心事に近い
- 「他の人の答えが気になる」と思わせる

## 質問パターン

- **選択系**: 「AとB、どっちが好きですか？」
- **経験系**: 「〜したことある人、いますか？」
- **意見系**: 「〜についてどう思いますか？」
- **完成系**: 「〜といえば私は→（答えを書く）」

## 出力形式

```
## 質問投稿: 「〜」テーマ

### パターン1（選択系）
〜

### パターン2（経験系）
〜

### パターン3（意見系）
〜
```

$ARGUMENTS にテーマ・ターゲット層を渡してください。
