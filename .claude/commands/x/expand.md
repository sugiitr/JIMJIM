# /x/expand — 投稿を膨らませて深みを出す

短いアイデア・メモ・一言をバズる投稿・スレッドに展開する。

## 手順

1. 元のアイデア・メモのエッセンスを確認する
2. 展開の方向性（深掘り・具体例・ストーリー化）を選ぶ
3. 単ツイート版・スレッド版の両方を生成する
4. 拡張した内容の強みを解説する

## 展開パターン

- **深掘り**: 「なぜ？」を3回繰り返す
- **具体化**: 抽象論に実体験・数字を足す
- **逆張り**: 反論を先取りして論理強化
- **ストーリー化**: before→after形式に変換
- **リスト化**: 分解して「〇つのポイント」へ

## 出力形式

```
## 元のアイデア
「〜」

## 単ツイート版 (展開済み)
〜

## スレッド版 (3〜5ツイート)
1/3: 〜
2/3: 〜
3/3: 〜
```

$ARGUMENTS に展開したい短いアイデア・メモ・一言を渡してください。
