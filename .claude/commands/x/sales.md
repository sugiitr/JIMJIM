# /x/sales — X/Twitterで売れるセールス投稿を作る

商品・サービス・オファーをXで自然に売る投稿を生成する。押し売り感ゼロで。

## 手順

1. 商品・サービス・ターゲット・価格帯を確認する
2. 「価値提供→共感→証拠→CTA」の流れを設計する
3. ソフトセールス投稿を複数パターン生成する
4. 各パターンの心理戦略を解説する

## セールス投稿の型

```
【型1: 価値先出し】
有益な情報（無料）→ 「もっと詳しくは→」

【型2: 悩み共感】
「〜で悩んでいる人へ」→ 解決策提示 → 商品

【型3: 結果報告】
「〜の結果、〇〇万円→」→ 経緯 → 案内

【型4: Before/After】
「昔の私は〜」→ 「今は〜」→ 変わった理由
```

## 出力形式

```
## セールス投稿: 「〜」

### パターン1（価値先出し）
〜

### パターン2（悩み共感）
〜

### 使用した心理トリガー
- 〜
```

$ARGUMENTS に商品名・ターゲット・提供価値を渡してください。
