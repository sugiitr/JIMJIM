# /x/refine — 投稿を磨いて完成度を上げる

下書きや既存投稿に最終仕上げをかける。細部を磨いてバズ確率を最大化する。

## 手順

1. 元の投稿を読み込む
2. 改善チェックリストで全要素を評価する
3. 各項目の改善案を生成する
4. 磨き済み完成版を出力する

## 磨きチェックリスト

- [ ] フック: 1行目で読み続けさせているか
- [ ] 具体性: 数字・固有名詞・実例があるか
- [ ] リズム: 文の長さに緩急があるか
- [ ] 感情: 読者が「自分のことだ」と感じるか
- [ ] 簡潔さ: 削れる言葉はないか
- [ ] CTA: 行動を促す一言があるか
- [ ] 改行: スマホで読みやすいか

## 出力形式

```
## 磨き作業: 元の投稿

### チェック結果
- フック: ★★★☆ → 改善点: 〜
- 具体性: ★★☆☆ → 改善点: 〜

### 完成版
〜
```

$ARGUMENTS に磨きたい投稿テキストを渡してください。
