# /x/lp — LPに誘導するX投稿を作る

ランディングページ・セールスページへの自然な誘導投稿を生成する。

## 手順

1. LP・商品・ターゲットを確認する
2. 「価値のチラ見せ」戦略を設計する
3. クリックしたくなる誘導投稿を生成する
4. A/Bテスト用に複数パターン作成する

## 誘導投稿の原則

- LPの内容をすべて書かない（クリックする理由を残す）
- 「続きはプロフのリンクから」より具体的なベネフィットで誘導
- 数字・限定・期限で緊急性を演出
- 読んだ人が「これ自分のことだ」と感じるターゲティング

## 出力形式

```
## LP誘導投稿: 「〜」

### パターン1（ベネフィット訴求）
〜
↓
🔗 [プロフのリンクへ]

### パターン2（限定・緊急性）
〜

### パターン3（ストーリー型）
〜
```

$ARGUMENTS にLPの内容・商品名・ターゲット・特典（あれば）を渡してください。
