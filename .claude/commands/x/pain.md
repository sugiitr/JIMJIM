# /x/pain — ペインポイントを刺す投稿を作る

ターゲットの痛み・悩み・フラストレーションに深く共感し、解決を示す投稿を生成する。

## 手順

1. ターゲット・商品・解決できる悩みを確認する
2. ペインポイントを深掘り（表面→本質→感情）する
3. 共感→問題提起→解決策の流れで投稿を生成する
4. 「わかってる感」の強さを確認する

## ペインの3層

```
【表面のペイン】目に見える不満
↓
【中間のペイン】その裏にある不安・恐れ
↓
【深層のペイン】アイデンティティへの脅威
```

## 共感→解決の型

```
「〜な状況、ありませんか？」
「〜でも〜でもない。本当の問題は〜」
「私も同じでした。変わったのは〜」
「〜を試してみてください」
```

## 出力形式

```
## ペイン投稿: 「〜ターゲット」

### ペイン分析
- 表面: 〜
- 中間: 〜
- 深層: 〜

### 投稿案
〜
```

$ARGUMENTS にターゲット・悩み・解決策（商品/ノウハウ）を渡してください。
