# /x/trend — X/Twitterのトレンドを調査する

現在のトレンドトピック・急上昇ハッシュタグ・話題のコンテンツを調査・分析する。

## 手順

1. 対象ジャンル・地域・期間を確認する
2. トレンドの背景・原因を調査する
3. 関連する投稿パターンを分析する
4. 乗っかり方・コンテンツ機会を提案する

## 出力形式

```
## トレンド調査: 「〜」

### なぜ今バズっているか
〜

### 関連ハッシュタグ
#〜 (件数: 〜)

### バズり投稿パターン
1. 〜
2. 〜

### コンテンツ機会
- 今すぐ使えるアングル: 〜
- 差別化ポイント: 〜
- 賞味期限: 〜
```

$ARGUMENTS にトレンドを調べたいジャンル・キーワード・業界を渡してください。
