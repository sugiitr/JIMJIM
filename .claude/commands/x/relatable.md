# /x/relatable — 「わかる！」を量産する共感投稿を作る

フォロワーが「これ自分のことだ」「わかりすぎる」と感じる共感投稿を生成する。

## 手順

1. ターゲット層の日常・悩み・あるあるを確認する
2. 「言語化されていない感情」を掘り起こす
3. 「わかる！」「あるある」「これすぎる」ネタを生成する
4. 保存・引用リツイートされやすい形に仕上げる

## 共感投稿の型

- **あるある系**: 「〜な人あるある：〜」
- **心の声系**: 「みんな思ってるけど言わないこと：〜」
- **あの感覚系**: 「〜するときの罪悪感、ありません？」
- **状態描写系**: 「〜なとき → （詳細な状況描写）」
- **共犯系**: 「私だけかな？〜」

## 出力形式

```
## 共感投稿: 「〜ターゲット」

### パターン1（あるある）
〜

### パターン2（心の声）
〜

### パターン3（状態描写）
〜

### 期待エンゲージメント
- いいね要因: 〜
- 保存要因: 〜
```

$ARGUMENTS にターゲット層・テーマ・あるあるのタネを渡してください。
