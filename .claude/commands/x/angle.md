# /x/angle — 同じテーマで異なる切り口を探す

飽きられた話題でも新鮮に見せる、差別化されたアングル（切り口）を生成する。

## 手順

1. テーマ・ジャンルを確認する
2. すでに多い「ありきたりアングル」を特定する
3. 差別化できる新しいアングルを10個生成する
4. 各アングルで投稿冒頭1行を書く

## アングル生成の手法

- **逆張り**: 常識と真逆のことを言う
- **ニッチ化**: 対象を絞り込む（「20代女性起業家向け」等）
- **時間軸変換**: 「過去/未来」視点で見る
- **業界横断**: 別ジャンルの視点を持ち込む
- **数値化**: 感覚的な話を数字で語る
- **比較**: 有名人・有名事例と比べる

## 出力形式

```
## アングル探索: 「〜」テーマ

### よくあるアングル（避けるべき）
- 〜

### 新しいアングル10案
1. 〜 → フック: 「〜」
2. 〜 → フック: 「〜」
...

### おすすめTOP3（差別化度）
1. 〜（理由: 〜）
```

$ARGUMENTS にテーマ・ジャンル・発信ジャンルを渡してください。
