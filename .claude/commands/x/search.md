# /x/search — X/Twitterを検索・調査する

指定したキーワード・トピックについてX/Twitter上のトレンド・投稿・文脈を調査する。

## 手順

1. 検索クエリを整理する
2. 関連するハッシュタグ・アカウント・キーワードを特定する
3. 検索戦略（演算子活用）を提示する
4. 調査すべき観点をリストアップする

## 出力形式

```
## 検索クエリ
メインキーワード: "〜"
関連キーワード: 〜, 〜, 〜
推奨ハッシュタグ: #〜, #〜

## 検索演算子の例
〜 from:〜
〜 min_faves:100
〜 lang:ja

## 調査観点
- ユーザーの反応・感情
- 拡散している投稿のパターン
- 批判・懸念点
```

$ARGUMENTS に検索したいトピック・キーワードを渡してください。
