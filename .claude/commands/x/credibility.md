# /x/credibility — 信頼性を高める投稿を作る

「この人は信頼できる」「嘘をついていない」と感じさせる投稿を生成する。

## 手順

1. 信頼を築きたい文脈・ターゲットを確認する
2. 信頼性を構成する要素（透明性・一貫性・実績・失敗談）を設計する
3. 信頼度を上げる投稿を複数生成する
4. 長期的な信頼構築の投稿計画を提案する

## 信頼性の4要素

1. **透明性**: 失敗・弱みも開示する
2. **一貫性**: 言っていることとやっていることが一致
3. **実績**: 具体的な成果・数字
4. **第三者評価**: 顧客の声・メディア掲載

## 信頼投稿の型

```
【失敗開示型】「私が〜に失敗した話」（人間味）
【プロセス公開型】「今日やったこと：〜」（透明性）
【顧客の声型】「〜さんからDMをいただいた」（社会的証明）
【一貫性型】「ずっと言い続けている〜の話」（信念）
```

## 出力形式

```
## 信頼性構築投稿: 「〜」テーマ

### 失敗開示型
〜

### プロセス公開型
〜

### 社会的証明型
〜
```

$ARGUMENTS に信頼を築きたい文脈・自分のジャンル・実体験を渡してください。
