# /x/offer — 魅力的なオファーをX投稿で伝える

商品・サービス・コンテンツのオファーを、断れない形でX投稿に落とし込む。

## 手順

1. オファー内容・ターゲット・価格・期限を確認する
2. 「価値 > 価格」を明確に見せる構成を設計する
3. オファー投稿を生成する
4. スタック（価値の積み上げ）表現を活用する

## オファーの構成要素

- **何が手に入るか**: 成果・変化・ベネフィット
- **なぜ今か**: 限定・期限・希少性
- **証拠**: 実績・実例・声
- **リスクゼロ化**: 保証・返金・無料体験
- **行動**: 申し込み方法・次のステップ

## 出力形式

```
## オファー投稿: 「〜」

### 本文
〜

### 価値スタック表示
✅ 〜（通常価格: 〇円）
✅ 〜（通常価格: 〇円）
合計〇円相当 → 今だけ〇円

### A/Bパターン
〜
```

$ARGUMENTS にオファー内容・価格・期限・ターゲットを渡してください。
