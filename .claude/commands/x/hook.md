# /x/hook — 引きの強いフックを生成する

投稿の冒頭1行（フック）をバリエーション豊富に生成する。フックが9割。

## 手順

1. テーマ・メッセージ・ターゲットを確認する
2. 10種類のフックパターンを適用する
3. 10〜15個のフック案を生成する
4. 各フックの狙い・心理効果を解説する

## フックパターン10選

1. **数字**: 「〇〇個の〜」
2. **逆説**: 「〜は間違いだった」
3. **暴露**: 「誰も言わない〜」
4. **警告**: 「〜してはいけない」
5. **共感**: 「〜な人は正直に手を挙げて」
6. **好奇心**: 「なぜ〜なのか？」
7. **約束**: 「〇分で〜できる」
8. **挑戦**: 「〜できないなら読んで」
9. **ストーリー**: 「〇年前、私は〜だった」
10. **断言**: 「〜は必ず〜する」

## 出力形式

```
## フック生成: 「〜」テーマ

1. 〜（数字系）
2. 〜（逆説系）
...
10. 〜（断言系）
```

$ARGUMENTS にフックを作りたいテーマ・メッセージを渡してください。
