# /x/persona — X発信ペルソナを設計する

自分のXアカウントのキャラクター・ポジション・発信スタイルを設計する。

## 手順

1. 自分のバックグラウンド・強み・発信ジャンルを確認する
2. ターゲットフォロワー像を定義する
3. ペルソナ設計書を生成する
4. ペルソナに基づく発信ガイドラインを提示する

## ペルソナ設計の5要素

1. **ポジション**: 業界内での立ち位置・差別化
2. **人格**: 性格・口調・価値観
3. **コンテンツの柱**: 発信するテーマ3〜5本
4. **ターゲット**: 理想のフォロワーの解像度
5. **禁忌**: 絶対に発信しないこと・スタイル

## 出力形式

```
## Xペルソナ設計書

### ポジション
「〜の人が〜について発信するアカウント」

### 人格・口調
〜

### コンテンツの柱
1. 〜（週〇回）
2. 〜（週〇回）
3. 〜（週〇回）

### 理想フォロワー
- 年齢: 〜
- 悩み: 〜
- 目標: 〜

### 発信してはいけないこと
- 〜
```

$ARGUMENTS に自分のバックグラウンド・強み・発信したいジャンルを渡してください。
