# /x/cta — 行動を促すCTAを生成する

読者に「フォロー・保存・クリック・購入」などの行動を起こさせるCTAを生成する。

## 手順

1. 促したいアクション・文脈を確認する
2. 摩擦を最小化したCTA文を複数生成する
3. 強度別（ソフト〜ハード）に分類する
4. 最適な配置・タイミングを提案する

## CTAの種類

- **エンゲージメント系**: いいね・RT・返信
- **フォロー系**: フォロー・通知ON
- **保存系**: ブックマーク・スクショ保存
- **誘導系**: プロフィール・リンクへ
- **購入系**: DM・購入・申し込み

## 出力形式

```
## CTA生成: 「〜」アクション

### ソフトCTA（低摩擦）
- 〜
- 〜

### ミドルCTA
- 〜
- 〜

### ハードCTA（高誘導）
- 〜

### 最適配置アドバイス
〜
```

$ARGUMENTS に促したいアクションと文脈（どんな投稿の末尾か）を渡してください。
