# /x/positioning — X上でのポジショニングを設計する

競合と差別化された、自分だけの「Xでの立ち位置」を設計する。

## 手順

1. 自分の強み・経験・ジャンルを確認する
2. 競合アカウントの立ち位置をマッピングする
3. 空白地帯を特定し、自分のポジションを決める
4. ポジションを一言で表すキャッチコピーを生成する

## ポジショニングの軸

```
【軸1: 誰向けか】
ビギナー ←→ プロ・上級者

【軸2: スタイル】
感情・共感系 ←→ ロジック・データ系

【軸3: 内容】
広く浅く ←→ 狭く深く

【軸4: トーン】
フレンドリー ←→ 権威・エキスパート
```

## 出力形式

```
## ポジショニング設計: 「〜」ジャンル

### 競合マッピング
| アカウント | ターゲット | スタイル | 強み |
|-----------|-----------|---------|------|
| @〜 | 〜 | 〜 | 〜 |

### あなたのポジション
- 立ち位置: 〜
- 差別化ポイント: 〜
- キャッチコピー: 「〜」

### このポジションで勝てる理由
〜
```

$ARGUMENTS に自分の強み・ジャンル・競合アカウント名を渡してください。
