# /branch — mainを守る門番

## 手順

1. `git rev-parse --abbrev-ref HEAD` で現在のブランチを確認
2. `git status` で未コミットの変更を確認。変更があれば:
   - `.env`、秘密情報、一時ファイル等が含まれていないか確認
   - ファイル名を指定して `git add`
   - 現在のブランチにコミットしてから次へ進む
3. main以外なら「ブランチ: {現在のブランチ名}」と報告して終了
4. mainなら、ユーザーのプロンプト原文から英語slugを生成し `git checkout -b {slug}` で作成

## ルール

- mainへの直接コミットは**禁止**
- ブランチ名はユーザーの指示内容を表す英語slug（例: `fix-date-parse-bug`）
- コミットメッセージにはユーザーのプロンプト原文を含める
